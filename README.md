# Muhammed Irshath Portfolio

A modern, responsive portfolio website showcasing my skills, projects, and experience as an Information and Communications Technology student at Singapore Institute of Technology.

## Features

- **Dark/Light Mode**: Toggle between dark and light themes with persistent preference storage
- **Responsive Design**: Optimized for all screen sizes and devices
- **Interactive Skills Section**: Hover effects displaying technology icons
- **Project Showcase**: Featured projects with GitHub repository links
- **Contact Form**: Functional contact form using EmailJS integration
- **Modern Tech Stack**: Built with React.js and Vite for optimal performance

## Tech Stack

- React.js
- Vite
- CSS Modules
- EmailJS
- React Icons

## Installation

```bash
git clone https://github.com/MuhammedIrshath49/portfolio.git
cd portfolio
npm install
```

## Development

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view the portfolio in development mode.

## Build

```bash
npm run build
```

## Deployment

This portfolio can be deployed on:
- Netlify
- Vercel
- GitHub Pages
- Any static hosting service
